# Platform Fighter Template

A simple 2D platformer fighting game template built with HTML, CSS, and JavaScript.

## Features

- Two-player local controls
- Jumping, movement, and light attack mechanics
- Simple platform stage and gravity
- Health display and win state

## Controls

- Player 1: `A` / `D` to move, `W` to jump, `S` to attack
- Player 2: `←` / `→` to move, `↑` to jump, `↓` to attack

## Run

1. Open `index.html` in a browser.
2. Play locally with two players.
3. Click `Restart` to reset the match.
